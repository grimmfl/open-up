## Open Up - An Open Voice Chat
Open Up is a free, privacy-first and open-source voice chat based on WebRTC:

| [Windows](https://github.com/grimmfl/open-up/releases/latest/download/OpenUp.exe) | ✅ Yes        |

### Notes
The app is not signed, as this costs at least 70$ per year per OS.

Because of that, you will probably receive a Windows Defender warning on Windows.

On Mac, there will likely be an error as well. You can run the file after executing the command below in a terminal. /path/to/file will usually be /Applications/OpenUp.app
```shell
xattr -d com.apple.quarantine /path/to/file
```
